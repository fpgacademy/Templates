This file includes brief descriptions on all the file in this directory.

default.tex	: includes some default commands, such as publish date
preamble.tex	: includes necessary settings and packages for the document template
preamble_ip.tex	: includes extra settings and packages for the IP documents
doxygen.sty	: style file for doxygen generated latex files
copyright.tex	: The copyright message to be inserted at the end of all Docs
